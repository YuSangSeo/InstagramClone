## InstagramClone

Instagram clone with Express + prisma + React and React native

## User Stories

- [x] Create account
- [ ] Request Secret
- [ ] Confirm Secret (Login)
- [ ] See the feed
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the photo
- [ ] Edit my profile
- [ ] Upload a photo 
- [ ] Edit the photo (Delete)
